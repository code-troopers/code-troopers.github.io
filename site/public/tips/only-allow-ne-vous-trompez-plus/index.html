only-allow : ne vous trompez plus

  • npm
  • yarn

Ça vous arrive de switcher de projet et de vous tromper entre npm et yarn ? Plus maintenant, grâce à only-allow !

Ajoutez juste une commande en preinstall (plus d’info sur le readme) et c’est parti

{
  "scripts": {
    "preinstall": "npx only-allow npm"
  }
}
